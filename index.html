Configurazione API

Come ottengo la mia API Key? (Clicca per vedere)

1. Vai su Google AI Studio.

2. Accedi con il tuo account Google.

3. Clicca su "Get API key" nella barra laterale sinistra.

4. Clicca su "Create API key" e copia la stringa alfanumerica.

5. Incollala nel box sopra e clicca "Salva".
